# Validierung

Automatisch geprüft beim Erstellen:

- `index.html` vorhanden
- `app.js` vorhanden
- `styles.css` vorhanden
- `sources.json` vorhanden
- `data/basics.json` vorhanden
- `data/news.json` vorhanden
- `.github/workflows/update-and-deploy.yml` vorhanden
- `.nojekyll` vorhanden
- JSON-Dateien parsebar
- Workflow für 2x tägliche Aktualisierung vorhanden

Hinweis: Die automatische Aktualisierung hängt davon ab, ob GitHub Actions im Ziel-Repository aktiviert ist und ob die überwachten Quellen erreichbar sind.
